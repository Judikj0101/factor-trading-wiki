# Low Volatility faktor

## Definíció

Alacsonyabb volatilitású (beta, std dev) részvények szisztematikusan felülteljesítik a magasabb volatilitásúakat kockázattal korrigálva. Ez ellentmond a CAPM alapfeltevésének (magasabb kockázat = magasabb hozam).

## Mérőszámok

- 1–3 éves standard deviáció
- Downside standard deviation
- Idiosyncratic return std dev
- Beta

## Prémium magyarázatok

Nincs hagyományos risk-based magyarázat — ez anomália a CAPM keretben:

| Magyarázat | Leírás |
|---|---|
| Lottery effect | Befektetők túlfizetnek a magas volatilitású "lottó" részvényekért |
| Overconfidence | Túlzott önbizalom magas beta részvényeknél |
| Leverage aversion | Intézményi befektetők nem használhatnak tőkeáttételt, ezért magas beta részvényekkel próbálnak magasabb hozamot elérni → túlfizetés |
| Benchmark constraint | Relatív hozam optimalizálás → a volatilitás nem "drága" a tracking error keretben |
